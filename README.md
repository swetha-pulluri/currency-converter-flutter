# 💱 Currency Converter App (Flutter)

A modern Flutter app to convert currencies in real-time.

---

## 🚀 Features
- Real-time currency conversion
- Currency swap functionality
- Clean and modern UI
- Provider state management
- Conversion history

## 🛠 Tech Stack

- Provider
- REST API
